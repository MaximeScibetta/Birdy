# Birdy ria
> Application react-native permettant d'encoder des oiseaux caputré ainsi que les modifier.

## Fonctionnalité

* Authentification par email et mot de passe
* Inscription email, mot de passe, nom complet
* Deconnexion
* Carte avec les lieux de captures
* Formulaire d'ajout d'oiseau capturé
    * Comment
    * Quand
    * Où
    * Nom latin
    * Nom
    * N° de bague
    * N° de série de la bague
    * Longueur alair
    * Taux de graisse
    * Sexe
    * Âge
* Formulaire de modification de la capture
* Encyclopédie
    * Son
    * Carte 
    * Information diverses
    * Lien vers page de la capture (site)
* Fil d'actualité des captures
* Profil d'utilisateur (nom, email)
    * Capture de l'utilisateur
* Recherche des utilisateurs

## Technologies

* React-native
* firebase
* Redux pour navigation
